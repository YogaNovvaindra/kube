# Tools

A collection of utility tools, development environments, and automation platforms.

## Subdirectories

- **[`archiving/`](tools/archiving)**: Distributed archiving tools (Archive Team Warrior).
- **[`authentik/`](tools/authentik)**: Identity and Access Management (SSO).
- **[`automation/`](tools/automation)**: Workflow automation tools (n8n, Changedetection).
- **[`cluster/`](tools/cluster)**: Cluster management tools (Semaphore, Syncthing, Netbird).
- **[`db/`](tools/db)**: Database management tools (pgAdmin, Bytebase, phpMyAdmin).
- **[`editor/`](tools/editor)**: Web-based editors and document tools (Outline, Excalidraw, BentoPDF).
- **[`git/`](tools/git)**: Git hosting and management (Gitea).
- **[`homepage/`](tools/homepage)**: Dashboard for the homelab.
- **[`network/`](tools/network)**: Network utilities (Speedtest).
- **[`security/`](tools/security)**: Security tools and password managers (Vaultwarden).
- **[`storage/`](tools/storage)**: File management and object storage (FileBrowser, RustFS).
- **[`utilities/`](tools/utilities)**: General developer utilities (IT-Tools).
