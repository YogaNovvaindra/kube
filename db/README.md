# 🗄️ Database Configurations

This directory contains various database definitions, services, and credentials used throughout the cluster.

## 📂 Structure

- **`mariadb.yml`**: MariaDB instance configuration.
- **`postgres.yml`**: PostgreSQL instance configuration.
- **`valkey.yml`**: Valkey (Redis-compatible) instance configuration.
- **`db-backup.yml`**: Database backup configuration.
- **`pgadmin.yml`**: pgAdmin database management tool.
- **`phpmyadmin.yml`**: phpMyAdmin database management tool.
